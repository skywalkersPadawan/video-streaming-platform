# 🎬 Netstream

A Netflix-inspired video streaming platform built with modern web technologies.  
This project focuses on scalable frontend architecture, real-time data integration, and media streaming systems.

---

## 🚀 Features

- 🎥 Dynamic movie browsing using TMDB API
- 📂 Category-based rows (Trending, Top Rated, Genres)
- 🧠 Modal-based preview with movie details
- 🔐 Authentication (local session-based)
- 📺 Responsive Netflix-style UI

---

## 🏗 Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Node.js, Prisma
- **Data Source:** TMDB API
- **Streaming (WIP):** HLS (hls.js), DASH
